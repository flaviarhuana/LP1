# Linguagem à Programação
Projeto EduConnect, apresentado como requisito para aprovação na disciplina.

Discente: Flávia Rhuana Pereira Sales

O projeto EduConnect compõe a segunda avaliação da disciplina e tem como objetivo demonstrar, por meio de um sistema funcional, a aplicação dos conceitos fundamentais de Programação Orientada a Objetos. Por tratar-se de um projeto introdutório, seu escopo foi deliberadamente simplificado, apresentando as seguintes limitações:

- Cada aluno pode estar vinculado a apenas um curso por vez;
- As notas são atribuídas ao aluno, e não ao curso especificamente;
- As presenças também são registradas diretamente no aluno, sem distinção entre cursos;
- A emissão de certificado considera um único conjunto de notas e uma única carga horária;
- Todo o fluxo do sistema é estruturado sob a lógica 1 aluno → 1 curso.
